# TestApplication

Foobar is a Python library for dealing with word pluralization.

## Installation

Installing all the dependencies

```bash
yarn install
```

Script to install pods and run iOS App

```bash
yarn pod
```

or if pods are already installed

```bash
yarn ios
```

To install App in Android

```bash
yarn android
```
