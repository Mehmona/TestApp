# TestApplication

## Installation

Installing all the dependencies

```bash
yarn install
```

### iOS

if got any error, Try running with XCode

Script to install pods and run iOS App

```bash
yarn pod
```

or if pods are already installed

```bash
yarn ios
```

### Android

To install App in Android

```bash
yarn android
```
